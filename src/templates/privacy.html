Privacy Policy

MACM Agent Tools API

Last Updated: October 9, 2025

1. Introduction

Welcome to MACM Agent Tools API. This privacy policy explains how we collect, use, and protect your information when you use our Multi-purpose Application Composition Model (MACM) API services.

2. Information We Collect

2.1 Automatically Collected Information

  • API request logs and timestamps
  • IP addresses and user agent information
  • Request parameters and response data
  • Error logs and debugging information

2.2 Data You Provide

  • Model data submitted for validation
  • Cypher queries and database interactions
  • Catalog information and configurations

3. How We Use Your Information

We use the collected information for:

  • Providing API services and model validation
  • Monitoring system performance and reliability
  • Debugging and improving our services
  • Ensuring security and preventing abuse
  • Generating anonymous usage statistics

4. Data Storage and Security

We implement appropriate security measures to protect your data:

  • Encrypted data transmission (HTTPS)
  • Secure database storage with access controls
  • Regular security audits and updates
  • Limited access to authorized personnel only

5. Data Retention

We retain your data for the following periods:

  • API logs: 30 days
  • Error logs: 90 days
  • User-submitted data: Until deletion is requested
  • Anonymous analytics: Indefinitely

6. Data Sharing

We do not sell, trade, or rent your personal information to third parties. We may share data in the following circumstances:

  • With your explicit consent
  • To comply with legal obligations
  • To protect our rights and security
  • In anonymized form for research purposes

7. Your Rights

You have the right to:

  • Access your personal data
  • Request correction of inaccurate data
  • Request deletion of your data
  • Object to processing of your data
  • Data portability

8. Cookies and Tracking

Our API may use minimal tracking mechanisms for:

  • Session management
  • Performance monitoring
  • Security purposes

9. Third-Party Services

We may integrate with third-party services such as:

  • Neo4j database services
  • Monitoring and analytics tools
  • Cloud infrastructure providers

10. Changes to This Policy

We may update this privacy policy from time to time. We will notify users of significant changes through our API documentation or direct communication.

11. Contact Information

If you have any questions about this privacy policy or our data practices, please contact us:

  • Project: MACM Agent Tools
  • Repository: https://github.com/umb-hub/MACM-Agent-Tools
  • Issues: Please create an issue on our GitHub repository

12. Compliance

This privacy policy is designed to comply with applicable data protection regulations, including GDPR and other relevant privacy laws.
